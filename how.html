How It Worksयह कैसे काम करता है

Learn how PhishGuard uses Machine Learning to protect you from phishing URLs and malicious emails. जानें कि PhishGuard फ़िशिंग URLs और दुर्भावनापूर्ण ईमेल से बचाने के लिए मशीन लर्निंग का उपयोग कैसे करता है।

⚡ Real-Time Detection⚡ रियल-टाइम पहचान 🧠 XGBoost ML Model🧠 XGBoost ML मॉडल 📧 Email Analysis📧 ईमेल विश्लेषण 📊 SHAP Explanations📊 SHAP स्पष्टीकरण

1️⃣ Enter a URL or Email1️⃣ URL या ईमेल दर्ज करें

Paste any suspicious URL or full email content into the scanner. PhishGuard accepts both — switch between URL and Email mode using the toggle buttons on the home page. कोई भी संदिग्ध URL या पूरी ईमेल सामग्री स्कैनर में पेस्ट करें। PhishGuard दोनों को स्वीकार करता है — होम पेज पर टॉगल बटन का उपयोग करके URL और ईमेल मोड के बीच स्विच करें।

2️⃣ ML Model Analyses It2️⃣ ML मॉडल विश्लेषण करता है

For URLs, 50 features are extracted — domain structure, entropy, suspicious keywords, TLD, file extensions and more. For emails, TF-IDF converts the text into 10,000 word-weight features. Both are fed into a trained XGBoost model. URLs के लिए, 50 विशेषताएं निकाली जाती हैं — डोमेन संरचना, एन्ट्रॉपी, संदिग्ध कीवर्ड, TLD, फ़ाइल एक्सटेंशन आदि। ईमेल के लिए, TF-IDF टेक्स्ट को 10,000 वर्ड-वेट फीचर में बदलता है। दोनों को प्रशिक्षित XGBoost मॉडल में डाला जाता है।

3️⃣ Get Instant Results3️⃣ तुरंत परिणाम पाएं

You receive a verdict — Safe, Phishing, Malware, or Defacement — with a confidence score, class probabilities, model metrics, and SHAP-based explanations. आपको एक निर्णय मिलता है — सुरक्षित, फ़िशिंग, मैलवेयर, या डिफेसमेंट — विश्वास स्कोर, वर्ग संभावनाओं, मॉडल मेट्रिक्स और SHAP-आधारित स्पष्टीकरण के साथ।

🔗 URL Model🔗 URL मॉडल

Trained on 650,000+ real URLs. Classifies into 4 categories — Benign, Phishing, Malware, and Defacement — with 96%+ accuracy using XGBoost. 650,000+ वास्तविक URLs पर प्रशिक्षित। XGBoost का उपयोग करके 96%+ सटीकता के साथ 4 श्रेणियों में वर्गीकृत करता है — Benign, Phishing, Malware और Defacement।

📧 Email Model📧 ईमेल मॉडल

Trained on 18,000+ real phishing and safe emails. Uses TF-IDF vectorization with XGBoost to detect phishing email content with 97%+ accuracy. 18,000+ वास्तविक फ़िशिंग और सुरक्षित ईमेल पर प्रशिक्षित। XGBoost के साथ TF-IDF वेक्टराइज़ेशन का उपयोग करके 97%+ सटीकता के साथ फ़िशिंग ईमेल सामग्री का पता लगाता है।

🔍 SHAP Explanations🔍 SHAP स्पष्टीकरण

Every URL verdict comes with SHAP-based reasons — showing which features raised or lowered the risk score, so you understand exactly why a URL was flagged. हर URL निर्णय के साथ SHAP-आधारित कारण आते हैं — जो दिखाते हैं कि किन विशेषताओं ने जोखिम स्कोर को बढ़ाया या घटाया, ताकि आप समझ सकें कि URL को क्यों फ्लैग किया गया।
